# Monitoramento Vacina Pentavalente

Painel Streamlit para análise de cobertura de vacinação com validação de datas das doses.

## Como usar localmente

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Inicie o app:
   ```bash
   streamlit run penta.py
   ```
